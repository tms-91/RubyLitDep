How to write Hello World Scripts for command prompt

The basic scripts

Windows: 

@echo off
echo Hello World!
Unix: 

#!/bin/sh
echo Hello World!

But we cant see anything when launching it!

Oh no! The Window closes to fast! lets fix this. Windows: 

pause
Unix: 

read -p "Press [Enter] key to exit..."

I want to say my name!


@echo off
echo Hello $.$-name-$.$!
pause

			

#!/bin/sh
echo Hello $.$-name-$.$!
read -p "Press [Enter] key to exit..."
